Wtyczka umożliwia klientom zakup produktów ze sklepów WooCommerce z wykorzystaniem opcji leasingu i rat za pośrednictwem PKO Leasing.
 
== Description ==
Wtyczka PKO Leasing integruje opcję leasingu i płatności ratalnych ze sklepami WooCommerce, umożliwiając klientom finansowanie zakupów za pośrednictwem PKO Leasing.

== Installation ==
1. Prześlij wtyczkę do katalogu `wp-content/plugins`.
2. Aktywuj wtyczkę w panelu administracyjnym WordPress.
3. W ustawieniach wtyczki wprowadź dane uwierzytelniające połączenie udostępnione przez PKO Leasing.
